# 项目简介

## 核心功能
- 基于YOLOv8的行人车辆目标跟踪计数检测
- 支持图片、视频及摄像头三种输入源的检测
- 提供完整源码、VOC2007数据集、训练好的模型及项目运行教程
- 内含3000张VOC2007数据集，支持7类目标检测（aeroplane, bicycle, bus, car, motorbike, person, train）
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测及结果查看
- 支持检测结果保存
- 支持自定义绘制检测边界线和检测区域，并记录目标数量
- 提供UI界面，支持选择模型、调整置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 行人车辆计数分析
- 交通监控与管理
- 安防监控
- 图像识别研究
- 计算机视觉教学

## 技术特点
- 基于YOLOv8目标检测算法
- 使用Python 3.9开发，依赖OpenCV、PyQt5、Ultralytics、Torch1.9等库
- 提供PyCharm+Anaconda或VSCode+Anaconda开发环境
- 采用图形化用户界面，操作简便
